# Code of Conduct

This repository expects respectful, technical, good-faith collaboration.

## Expected behavior

- Be direct without being hostile.
- Focus criticism on code, decisions, and behavior, not people.
- Assume mistakes before malice.
- Keep review discussions constructive and evidence-based.
- Respect project boundaries around secrets, security, and responsible disclosure.

## Unacceptable behavior

- Harassment, intimidation, or personal attacks
- Discriminatory or demeaning language
- Deliberate disruption of reviews, issues, or discussions
- Publishing private or security-sensitive information without permission

## Reporting

If you experience or observe unacceptable behavior, contact the repository maintainer through GitHub rather than escalating in a public thread when possible.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this code of conduct and may restrict participation for repeated or serious violations.
