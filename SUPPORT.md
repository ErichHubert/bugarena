# Support

## Getting help

- Use GitHub issues for bugs, usability problems, and feature requests.
- For security issues, use the process in `SECURITY.md`.

When opening an issue, include:

- what you were trying to do
- the exact command or workflow you ran
- the observed result
- the expected result
- relevant logs or error output with secrets removed

## Scope

This repository is infrastructure and reference material for an isolated agent workstation plus capability broker pattern. Support is best effort and there is no service-level guarantee.
